

Чтобы запустить:
- npm install
- npm start

Архитектура:
- components/Header -- шапка вместе с кнопкой для выпадающего списка.
- components/Main/Table -- портфель.
- components/Main/list - выпадающий список валюты.
- components/Main/list/ButtonBlock - Блок button и input из выпадающего списка валюты.

- components/Currency.ts -- файл со списком доступных валют(можете дополнить список)

Дополнительные пакеты:
-Redux/toolkit
-Mobx
